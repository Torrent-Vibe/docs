# Screenshots Directory

This directory contains screenshots used in the documentation.

## Naming Convention

- `main-interface.png` - Main application interface
- `interface-layout.png` - Interface layout diagram
- `download-page.png` - Official download page
- `installation-wizard.png` - Installation wizard screenshots
- `first-run-setup.png` - First run setup wizard
- `qbittorrent-connection.png` - qBittorrent connection setup
- `torrent-list.png` - Torrent list view
- `torrent-details.png` - Torrent details panel
- `categories-management.png` - Categories management interface
- `tags-management.png` - Tags management interface
- `search-filters.png` - Search and filters interface
- `statistics-dashboard.png` - Statistics dashboard
- `bandwidth-control.png` - Bandwidth control settings
- `general-settings.png` - General preferences
- `download-settings.png` - Download settings
- `advanced-options.png` - Advanced options
- `multi-server-setup.png` - Multi-server configuration
- `automation-setup.png` - Automation features
- `integrations-setup.png` - Third-party integrations
- `customization-options.png` - Customization settings

## Image Specifications

- **Format**: PNG (preferred) or JPG
- **Resolution**: 1920x1080 or higher for main screenshots
- **Quality**: High quality, clear and readable
- **Annotations**: Use arrows, highlights, or callouts where helpful
- **Consistency**: Maintain consistent styling across all screenshots

## Usage in Documentation

Screenshots are referenced in MDX files using relative paths:

```markdown
![Main Interface Screenshot](./screenshots/main-interface.png)
*Figure 1: Torrent Vibe Main Interface Overview*
```

## Adding New Screenshots

1. Place the image file in this directory
2. Update this README with the new screenshot name and description
3. Reference the screenshot in the appropriate MDX file
4. Ensure the image is optimized for web (under 500KB preferred)
