# Developer Documentation

Welcome to the Torrent Vibe developer documentation. This section provides comprehensive information for developers who want to integrate with, extend, or contribute to Torrent Vibe.

> **Tip**: If you're not a developer and just want to use Torrent Vibe, please refer to the [User Guide](../user-guide/) section.

## üöÄ Getting Started

### Development Setup

If you're interested in contributing to Torrent Vibe or building integrations:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Torrent-Vibe/Torrent-Vibe.git
   cd torrent-vibe
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start development server**
   ```bash
   pnpm dev:electron  # For Electron development
   # or
   pnpm dev          # For web development
   ```

See [Development Setup](contributing/development-setup) for detailed instructions.
