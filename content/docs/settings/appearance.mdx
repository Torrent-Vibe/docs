# Appearance Settings

The Appearance settings panel allows you to personalize the visual interface of Torrent Vibe, including themes, language, and UI customization options.

![Appearance Settings](/screenshots/TorrentVibeScreenshot_AppearanceSettings.jpeg)

## Theme Configuration

### Appearance Theme
Choose the visual theme for the application:

- **System**: Follows your operating system preference
  - **Light**: Uses light theme when OS is light
  - **Dark**: Uses dark theme when OS is dark
  - **Automatic**: Adapts to system changes

- **Light**: Always use light theme
  - **Colors**: White backgrounds, dark text
  - **Best for**: Bright environments
  - **Battery**: May use more battery on OLED screens

- **Dark**: Always use dark theme
  - **Colors**: Dark backgrounds, light text
  - **Best for**: Low-light environments
  - **Battery**: More efficient on OLED screens

### Theme Customization
Advanced theme options:

- **Custom Theme**: User-defined color schemes
  - **Accent Colors**: Choose primary colors
  - **Background**: Custom background images
  - **Transparency**: Adjust window transparency

## Language Settings

### Interface Language
Select the display language:

- **System Language**: Use operating system language
- **Available Languages**: English, and other supported languages
- **Right-to-Left**: Support for RTL languages
- **Updates**: Language packs can be updated

### Localization Options
Additional language preferences:

- **Date Format**: Local date formatting
- **Number Format**: Local number formatting
- **Time Format**: 12/24 hour display
- **Currency**: Local currency display

## UI Personalization

### Layout Options
Customize the interface layout:

- **Sidebar Position**: Left or right side
- **Panel Sizes**: Adjustable panel widths
- **Toolbar**: Show/hide toolbar buttons
- **Status Bar**: Customize status information

### Display Preferences
Control what information is shown:

- **Columns**: Choose visible columns in torrent list
- **Details**: Show/hide detailed information
- **Icons**: Enable/disable icon display
- **Animations**: Control UI animations

## Accessibility

### Visual Accessibility
Improve readability and usability:

- **Font Size**: Adjust text size
  - **Small**: Compact display
  - **Medium**: Standard size
  - **Large**: Enhanced readability
  - **Extra Large**: Maximum accessibility

- **High Contrast**: Enhanced color contrast
  - **Enabled**: Better visibility
  - **Disabled**: Standard contrast

### Interaction Options
Customize user interactions:

- **Keyboard Navigation**: Full keyboard support
- **Focus Indicators**: Visual focus cues
- **Tooltips**: Show/hide help tooltips
- **Context Menus**: Right-click menu options

## Advanced Customization

### Custom Styles
Apply custom CSS styling:

- **User Styles**: Custom CSS rules
- **Theme Overrides**: Modify existing themes
- **Component Styling**: Style specific UI elements
- **Import/Export**: Share custom themes

### Background Options
Personalize the application background:

- **Default**: Standard application background
- **Custom Image**: Use personal background image
- **Blur Effect**: Apply background blur
- **Opacity**: Adjust background transparency

## Performance Considerations

### Theme Performance
- **System Theme**: Minimal performance impact
- **Custom Themes**: Slight performance cost
- **Animations**: Can be disabled for better performance
- **Transparency**: May impact performance on older hardware

### Resource Usage
- **Light Theme**: Standard resource usage
- **Dark Theme**: Potentially better battery life
- **Custom Images**: Additional memory usage
- **Animations**: CPU usage for smooth transitions

- **Menu Bar**: Native macOS menu integration
- **Notifications**: Native notification center

### Platform-Specific Features
- **macOS**: Full system integration

## Troubleshooting

### Theme Issues

**Theme not applying:**
- Restart the application
- Check theme file integrity
- Reset to default theme

**Display problems:**
- Adjust DPI scaling
- Check graphics drivers
- Verify display resolution

**Performance issues:**
- Disable animations
- Use system theme
- Reduce transparency

### Language Problems

**Language not changing:**
- Restart application
- Check language pack installation
- Verify system locale

**Text display issues:**
- Check font availability
- Adjust font size
- Reset to default language
