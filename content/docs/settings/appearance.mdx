Control the app's visual appearance and interface language settings.

![Appearance](/screenshots/TorrentVibeScreenshot_AppearanceSettings.jpeg)

## Core Features

**Theme Selection**: Supports system, light, and dark theme modes with automatic system preference adaptation

**Language Settings**: Choose interface display language with multi-language support

## Important Notes

- Theme changes take effect immediately without app restart
- Language settings require page refresh for full application
- System theme automatically switches based on OS settings
