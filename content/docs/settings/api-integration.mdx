Torrent Vibe integrates with various external services to enhance your torrent management experience with metadata and private tracker support.

## Supported Integrations

### OMDb API (Movie Database)

**Purpose**: Fetch movie and TV show metadata
**Configuration**: [API Tokens → OMDb API Key](api-tokens.mdx)

**Features**:
- Automatic content identification
- Movie/TV show information
- Poster and artwork fetching
- Rating and review data

### Private Tracker Integration

**M-Team Support**
- **Purpose**: Access private torrent trackers
- **Configuration**: [API Tokens → M-Team](api-tokens.mdx)
- **Features**: Search private content, API integration
