Configure remote web access to Torrent Vibe for management from any device with a browser.

![Web UI Settings](/screenshots/TorrentVibeScreenshot_WebUISettings.png)

## Server Configuration

**Port**: Web interface listening port
- Default: 8080
- Range: 1-65535
- Ensure firewall allows access

**Use HTTPS**: Enable secure connections
- Enabled: Secure https:// access (recommended)
- Disabled: Standard http:// access

## SSL Certificate (HTTPS only)

**Certificate Path**: SSL certificate file path
- .pem or .crt format
- Self-signed or CA purchased

**Key Path**: Private key file path
- .key format
- Keep secure, never share

## Authentication

**Username**: Web UI login username
- Default: admin
- Use strong, unique username

**Password**: Web UI login password
- Use complex password
- Required for web access

## Access Control

**IP Whitelist**: Restrict access by IP
- Allow specific IP ranges
- Block unauthorized access

## Interface Options

**Alternative UI**: Enable alternative interface
- Modern interface option
- Different layout and features

**Bypass authentication for localhost**: Skip login for local access
- Enabled: No password for 127.0.0.1
- Disabled: Always require authentication
