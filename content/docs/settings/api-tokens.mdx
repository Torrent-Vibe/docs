Configure API keys for external services to enhance torrent discovery, metadata fetching, and AI-powered features.

![API Tokens Settings](/screenshots/TorrentVibeScreenshot_APITokensSettings.png)

## Discover Services

**OMDb API**: Movie/TV metadata discovery
- Free API key from [omdbapi.com](https://omdbapi.com)
- Automatic content identification for torrents

## AI Integration

**OpenAI API**: AI-powered content analysis
- API key from [platform.openai.com](https://platform.openai.com)
- Optional custom base URL and model selection
- Used for torrent categorization and enhancement

## Metadata Services

**TMDB API**: Enhanced movie/TV database
- Free API key from [themoviedb.org](https://themoviedb.org)
- Detailed content information and posters
