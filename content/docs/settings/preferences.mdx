# Preferences Settings

The Preferences settings panel allows you to configure application behavior, polling intervals, and system integration options.

![Preferences Settings](/screenshots/TorrentVibeScreenshot_PreferencesPollingSettings.png)

## Data Polling

### Polling Settings
Control how often Torrent Vibe checks for updates:

- **Data Polling**: Update frequency from server
  - **Interval**: How often to poll for changes
  - **Default**: 3 seconds
  - **Range**: 1-60 seconds
  - **Impact**: Real-time vs. server load

### Polling Behavior
Configure polling characteristics:

- **Adaptive Polling**: Adjust based on activity
  - **Enabled**: Poll more frequently when active
  - **Disabled**: Fixed polling interval
  - **Smart**: Balance responsiveness and efficiency

## Desktop Features

### File Association Repair
Fix torrent file handling:

- **Repair torrent file association**: Restore .torrent file links
  - **Purpose**: Ensure Torrent Vibe opens .torrent files
  - **Process**: Updates system file associations
  - **Scope**: Current user account
  - **Safety**: Non-destructive operation

### Association Status
Monitor file handling:

- **Current Association**: Shows current .torrent handler
- **Test Association**: Verify file opening works
- **Reset Association**: Clear custom associations
- **Default Application**: Set Torrent Vibe as default

## System Integration

### Startup Options
Configure application launch behavior:

- **Start with system**: Launch on boot
  - **Enabled**: Auto-start with operating system
  - **Disabled**: Manual launch only
  - **Location**: System startup folder

- **Start minimized**: Launch in background
  - **Enabled**: Start in system tray
  - **Disabled**: Show main window
  - **Notification**: Show startup notification

### System Tray
Control tray icon behavior:

- **Show tray icon**: Display system tray presence
- **Minimize to tray**: Hide window to tray
- **Tray notifications**: Show alerts in tray
- **Tray menu**: Quick access menu

## Notification Settings

### Alert Preferences
Configure notification behavior:

- **Download complete**: Notify when downloads finish
- **Error notifications**: Alert on download errors
- **Connection status**: Notify connection changes
- **Update available**: Alert on application updates

### Notification Style
Customize alert appearance:

- **System notifications**: Use OS notification system
- **In-app notifications**: Show within application
- **Sound alerts**: Play notification sounds
- **Custom sounds**: Choose alert sounds

## Performance Preferences

### Resource Usage
Control application resource consumption:

- **CPU priority**: Set process priority
  - **Low**: Minimal CPU usage
  - **Normal**: Standard priority
  - **High**: Faster processing

- **Memory management**: Control RAM usage
  - **Conservative**: Minimize memory use
  - **Balanced**: Standard memory usage
  - **Aggressive**: Maximum performance

## Privacy Options

### Data Collection
Control data sharing:

- **Usage statistics**: Anonymous usage reporting
  - **Enabled**: Help improve application
  - **Disabled**: No data collection
  - **Privacy**: Fully anonymous

- **Crash reporting**: Automatic error reporting
  - **Enabled**: Help fix bugs
  - **Disabled**: Manual reporting only
  - **Data**: Includes error details only

## Backup & Sync

### Configuration Backup
Preserve settings:

- **Auto-backup**: Automatic settings backup
  - **Enabled**: Regular backup creation
  - **Disabled**: Manual backup only
  - **Location**: Default backup directory

- **Backup frequency**: How often to backup
  - **Daily**: Backup every day
  - **Weekly**: Backup weekly
  - **Manual**: User-initiated only

### Settings Sync
Synchronize preferences:

- **Sync across devices**: Share settings
  - **Enabled**: Sync via account
  - **Disabled**: Local settings only
  - **Account**: Requires user account

### Language & Region
Localization settings:

- **Interface language**: Application language

## Troubleshooting

### Common Issues

**Polling too frequent:**
- Increase polling interval
- Enable adaptive polling
- Check server performance

**File associations broken:**
- Run repair tool
- Check system permissions
- Verify file type registration

**High resource usage:**
- Adjust CPU/memory settings
- Disable unnecessary features
- Check for background processes

### Reset Options

**Reset preferences:**
- Restore default settings
- Clear custom configurations
- Preserve essential data

**Factory reset:**
- Complete settings reset
- Clear all user data
- Fresh application state
