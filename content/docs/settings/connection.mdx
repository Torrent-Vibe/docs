Configure peer connection encryption, network ports, connection limits, and proxy settings for optimal torrent performance.

![Connection Settings](/screenshots/TorrentVibeScreenshot_ConnectionSettings.png)

## Peer Connection

**Encryption**: Control peer connection security
- Disabled: Maximum compatibility (less secure)
- Enabled: Require encrypted connections (recommended)
- Forced: Force outgoing encryption (maximum security)

**Anonymous Mode**: Enhanced privacy protection
- Disables peer IP exchange
- Reduces tracking capabilities
- May slightly impact download speeds

## Listening Port

**Port**: Network port for incoming connections
- Default: 6881 (BitTorrent standard)
- Range: 1024-65535
- Random: Generate available port automatically

**UPnP/NAT-PMP**: Automatic port forwarding
- Enabled: Auto-configure router (recommended)
- Disabled: Manual port forwarding required

## Connection Limits

**Global maximum connections**: Total peer connections
- Default: 500
- Increase for faster networks
- Decrease for limited bandwidth

**Maximum connections per torrent**: Per-torrent limit
- Default: 100
- Balance bandwidth across multiple torrents

## Network Settings

**Proxy Server**: Route connections through proxy
- Types: None, HTTP, SOCKS4, SOCKS5
- Authentication support
- Host and port configuration

**IP Filtering**: Block/allow specific IP ranges
- Import/export filter lists
- Auto-update capabilities
