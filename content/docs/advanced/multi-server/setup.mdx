# Multi-Server Setup

Torrent Vibe supports connecting to multiple qBittorrent instances simultaneously, allowing you to manage torrents across different servers, locations, or configurations. This guide covers setting up and managing multiple servers.

## Understanding Multi-Server

### Benefits of Multi-Server

#### Server Management
- **Centralized Control**: Manage all servers from one interface
- **Unified View**: See all torrents across servers
- **Bulk Operations**: Perform operations on multiple servers
- **Server Comparison**: Compare performance across servers

#### Use Cases
- **Home and Work**: Manage home and work servers
- **Multiple Locations**: Servers in different physical locations
- **Different Configurations**: Servers with different settings
- **Backup Servers**: Primary and backup server setups

### Server Types

#### Primary Server
- **Main Server**: Primary torrent management server
- **Full Features**: Access to all Torrent Vibe features
- **Master Configuration**: Main configuration source
- **Primary Monitoring**: Main monitoring and alerts

#### Secondary Servers
- **Additional Servers**: Secondary server instances
- **Limited Features**: Subset of features available
- **Configuration Sync**: Sync settings from primary
- **Backup Role**: Serve as backup when primary fails
