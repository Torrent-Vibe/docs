# Source Code Access

Torrent Vibe offers source code access for developers and advanced users who want to contribute to the project or customize the application.

## Getting Source Code Access

### License Key Requirement

To access the source code, you need a valid lifetime license key.

**Steps to obtain access**:

1. **Purchase Lifetime License**: [Buy for $9.99](https://www.creem.io/payment/prod_3MvxEew9LAj3kofPW6vqia)
2. **Receive License Key**: License key sent to your email instantly
3. **Download Full Version**: [Download from GitHub](https://github.com/Torrent-Vibe/Torrent-Vibe/releases)
4. **Note your details**: Keep your purchase email and GitHub username ready

![Get Source Code Access](/screenshots/TorrentVibeScreenshot_GetSourceCodeAccess.jpeg)

1. **Launch Access Dialog**

## Building from Source

### Clone Repository

```bash
git clone https://github.com/your-access-url/torrent-vibe.git
cd torrent-vibe
```

### Restore Dependencies

```bash
pnpm install
```

### Build Application

```bash
pnpm build
```

### Run Application

```bash
pnpm dev
```
