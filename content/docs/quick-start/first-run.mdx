## Initial Setup

### Empty State
When you first launch Torrent Vibe, you'll see an empty interface ready for your first torrent.

![Empty Interface](/screenshots/TorrentVibeScreenshot_MainEmptyInterface.jpeg)

The interface shows:
- **Add Button**: Blue "+" button to add new torrents
- **Drag & Drop Area**: Drop torrent files or magnet links here
- **Connection Status**: Green banner when successfully connected

### Connecting to qBittorrent

To connect to your qBittorrent server:

1. The application will prompt for connection details
2. Enter your server information:
   - **Host**: Server address (e.g., `127.0.0.1` or remote IP)
   - **Port**: Web UI port (default: `8080`)
   - **Username**: Your qBittorrent username
   - **Password**: Your qBittorrent password
   - **HTTPS**: Enable for secure connections (encrypts your connection to protect sensitive information)

![Connection Dialog](/screenshots/TorrentVibeScreenshot_qBittorrent_Connection_Dialog.jpeg)

### Connection Success
Once connected, you'll see a success notification and can begin managing torrents.

## Adding Your First Torrent

### Method 1: Add Button
1. Click the blue "+" button in the main interface
2. Choose to add from torrent file or magnet link
3. Configure download settings as needed

### Method 2: Drag & Drop
Simply drag torrent files or magnet links directly into the application window.

![Add Torrent Dialog](/screenshots/TorrentVibeScreenshot_AddTorrentDialog.png)

### Torrent Settings
When adding torrents, you can configure:
- **Save Path**: Where to store downloaded files
- **Category**: Organize torrents by category
- **Torrent Management**: Skip hash checking, sequential download, etc.
- **Start Options**: Begin download immediately or manually

## Basic Operations

### Monitoring Downloads
- View real-time download/upload speeds
- Track progress with visual progress bars
- Monitor peer connections and file status

![Download Progress](/screenshots/TorrentVibeScreenshot_DownloadingProgress.jpeg)

### Managing Torrents
- **Pause/Resume**: Control download states
- **Categories**: Organize by type (Movies, Software, etc.)
- **Priority**: Set download priorities
- **File Selection**: Choose specific files to download
