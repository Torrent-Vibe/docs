# Connecting to qBittorrent

Torrent Vibe connects to qBittorrent Web UI.

## Enable qBittorrent Web UI

1. Open qBittorrent → Tools → Preferences
2. Go to Web UI tab
3. Check "Enable Web User Interface"
4. Set username and password
5. Set port (default: 8080)

## Connect in Torrent Vibe

1. Launch Torrent Vibe
2. Enter connection details:
   - Host: `localhost` or IP
   - Port: `8080`
   - Username/password from qBittorrent

## Connection Types

- **Local**: `localhost` (same machine)
- **Network**: IP address (same network)
- **Remote**: Public IP (outside network, use HTTPS)

## Troubleshooting

- **Connection refused**: Check if qBittorrent is running and port is correct
- **Auth failed**: Verify username/password
- **Timeout**: Check network and reduce active torrents

## Multiple Servers

Add multiple qBittorrent instances in Settings → Server Management.
