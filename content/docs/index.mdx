# Welcome to Torrent Vibe

Torrent Vibe is a powerful, modern BitTorrent client built on the MonoTorrent library and .NET platform. It provides an intuitive interface for downloading, managing, and seeding torrent files with advanced features like multi-server support, API integrations, and extensive customization options.

![Main Interface](/screenshots/TorrentVibeScreenshot_MainInterface.jpeg)

- **Real-time Monitoring**: Live download/upload speeds and progress tracking
- **API Integrations**: OMDb, OpenAI, and private tracker support
- **Web UI Access**: Remote management through web interface
- **Cross-Platform**: Native macOS application with web capabilities

### Connecting to qBittorrent
To connect to your qBittorrent server:

1. Click the connection settings or use the setup wizard
2. Enter your server details (host, port, username, password)
3. Enable HTTPS for secure connections if needed
