# Server Management

Torrent Vibe supports connecting to multiple qBittorrent servers, enabling flexible torrent management across different locations and setups.

## Server Configuration

### Adding a Server

1. Go to Settings â†’ Servers
2. Click "+ Add" button
3. Enter server details:
   - **Server Name**: Descriptive identifier
   - **Host**: IP address or hostname
   - **Port**: Web UI port (default: 8080)
   - **Username**: qBittorrent username
   - **Password**: qBittorrent password

![Server Settings](/screenshots/TorrentVibeScreenshot_ServersSettings.png)

### Server Types

**Local Server**
- Host: `localhost` or `127.0.0.1`
- Port: `8080` (default)
- Use case: Same machine setup

**Network Server**
- Host: Local IP (e.g., `192.168.1.100`)
- Port: Configured port
- Use case: Same network access

**Remote Server**
- Host: Public IP or domain
- Port: Forwarded port
- Use case: External access

## Path Mapping

### Understanding Path Mapping

Path mapping translates file paths between your qBittorrent server and local system:

- **Server Path**: Location on qBittorrent server
- **Local Path**: Corresponding location on your machine
- **Purpose**: Access server files locally

### Setting Up Path Mapping

1. In Servers settings, find "Path Mapping" section
2. Click "+ Add mapping"
3. Configure:
   - **Server Path**: `/downloads/` (server location)
   - **Local Path**: `/Users/username/Downloads/` (local location)

### Common Mapping Scenarios

**Network Attached Storage (NAS)**
```
Server Path: /downloads/
Local Path: /Volumes/NAS/Downloads/
Purpose: Access NAS files locally
```

**External Drive**
```
Server Path: /media/
Local Path: /Volumes/External/
Purpose: USB drive access
```

**Cloud Storage**
```
Server Path: /cloud/
Local Path: /Users/username/Cloud/
Purpose: Cloud-synced files
```

## Troubleshooting

### Connection Issues

**Can't Connect**
- Verify server is running
- Check network connectivity
- Test Web UI access directly
- Check firewall settings

**Authentication Failed**
- Verify username/password
- Check account permissions
- Reset qBittorrent credentials
- Check for account lockout

**Path Mapping Problems**
- Verify paths exist
- Check permissions
- Test network access
- Update mapping configuration

### Performance Issues

**Slow Response**
- Check server load
- Monitor network latency
- Optimize server settings
- Consider server upgrade

**Sync Problems**
- Check network connectivity
- Verify server compatibility
- Review sync settings
- Check for conflicts
