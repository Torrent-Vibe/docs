# Interface Overview

Torrent Vibe has a modern interface for torrent management.

<img src="/screenshots/02-01-main-interface.png" alt="Main Interface Screenshot" />

## Main Layout

- **Header**: Search, settings, user menu
- **Sidebar**: Navigation, filters, categories  
- **Main Area**: Statistics, torrent list, details
- **Status Bar**: Connection status, system info

<img src="/screenshots/02-02-interface-layout.png" alt="Interface Layout Diagram" />

## Key Features

### Header Bar
- Global search and filters
- Add/pause/resume/delete torrents
- Settings and notifications

### Sidebar
- Dashboard, torrents by status
- Categories, tags, filters
- Statistics and preferences

### Main Content
- **Statistics**: Download/upload speeds, totals
- **Torrent List**: Name, size, progress, status, speed
- **Details Panel**: Torrent information and controls

## Controls
- Right-click menus for torrents and files
- Keyboard shortcuts (Ctrl+F search, Delete remove, etc.)
- Responsive design for desktop/tablet/mobile

## Customization
- Resize panels, show/hide columns
- Light/dark themes
- Notification settings
