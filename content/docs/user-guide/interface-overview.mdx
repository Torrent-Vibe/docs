Torrent Vibe features a modern, intuitive interface designed for efficient torrent management. The application provides comprehensive tools for downloading, monitoring, and organizing your torrent library.

![Main Interface](/screenshots/TorrentVibeScreenshot_MainInterface.jpeg)

## Main Interface Layout

### Primary Components

**Title Bar & Menu**
- Application title "Torrent Vibe"
- macOS window controls (close, minimize, maximize)
- Menu bar with File, Edit, View, Help options

**Toolbar**
- **Add Button** (Blue "+"): Add new torrents
- **Control Buttons**: Play, pause, stop torrent operations
- **Search Bar**: Filter and search torrents
- **Settings Gear**: Access application settings

**Main Content Area**
- **Torrent List**: Displays active downloads with progress
- **Status Information**: Speed, size, progress indicators
- **File Details**: Individual torrent information

**Right Panel**
- **Details Sidebar**: Torrent-specific information
- **Statistics**: Download/upload speeds, ratios
- **Peer Information**: Connected peers and trackers

## Settings Access

### Settings Panel
Access comprehensive configuration through the gear icon:

- **Downloads**: Path and behavior settings
- **Speed**: Bandwidth limits and scheduling
- **Connection**: Network and peer settings
- **BitTorrent**: Protocol configuration
- **Web UI**: Remote access setup
- **API Tokens**: External service integration
- **Appearance**: Theme and UI customization
- **Servers**: Multi-server management
- **Advanced**: Expert-level options
- **Preferences**: Polling and system integration

## Keyboard Shortcuts

### Common Shortcuts
- `Cmd/Ctrl + N`: Add new torrent
- `Cmd/Ctrl + F`: Focus search
- `Delete`: Remove selected torrent
- `Space`: Pause/resume torrent
- `Cmd/Ctrl + ,`: Open settings

### Navigation
- `Tab`: Move between panels
- `Arrow Keys`: Navigate torrent list
- `Enter`: Open torrent details
- `Esc`: Close dialogs
