Torrent Vibe features a modern, intuitive interface designed for efficient torrent management.

![Downloading Progress](/screenshots/TorrentVibeScreenshot_DownloadingProgress.jpeg)

## Main Interface Layout

### Primary Components

**Title Bar**
- Application title "Torrent Vibe"
- macOS window controls (close, minimize, maximize)

**Toolbar**
- **Add Button** (Blue "+"): Add new torrents
- **Control Buttons**: Play, pause, stop, delete selected torrents
- **System Information**: Free disk space, download/upload speeds
- **Search Bar**: Filter and search torrents
- **Settings Gear**: Access application settings

**Torrent Status Filters**
- Filter torrents by status: All, Downloading, Seeding, Completed, Paused, Error
- Shows count for each status category

**Main Content Area**
- **Torrent List**: Displays active torrents with columns for name, size, progress
- **Column Headers**: Name, Size, Progress (sortable)
- **Each Torrent**: Shows torrent name, size, category, and selection checkbox

**Right Panel (Details Sidebar)**
- **Torrent Summary**: Icon, full name, category, size
- **Navigation Tabs**: Information, Files, Peers, Settings
- **Progress Section**: Completion percentage, progress bar, pieces visualization
- **Overview Section**: Status, size, ratio, ETA
