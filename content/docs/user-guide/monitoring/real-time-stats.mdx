# Real-time Statistics

Torrent Vibe provides comprehensive real-time monitoring capabilities to track your torrent activity, performance metrics, and system health. This guide covers all monitoring features and how to use them effectively.

## Dashboard Overview

### Main Statistics Panel

#### Traffic Statistics
- **Download Speed**: Current download rate (updated every second)
- **Upload Speed**: Current upload rate (updated every second)
- **Total Downloaded**: Session download total
- **Total Uploaded**: Session upload total
- **Active Downloads**: Number of torrents currently downloading
- **Active Uploads**: Number of torrents currently uploading

#### System Statistics
- **CPU Usage**: Processor utilization percentage
- **Memory Usage**: RAM consumption
- **Disk I/O**: Read/write operations per second
- **Network Usage**: Network interface utilization
- **Free Disk Space**: Available storage space

### Quick Stats Cards

#### Torrent Overview
- **Total Torrents**: Total number of torrents in system
- **Downloading**: Torrents actively downloading
- **Seeding**: Torrents actively uploading
- **Completed**: Finished downloads
- **Paused**: Stopped torrents
- **Error**: Torrents with errors

#### Performance Metrics
- **Global Ratio**: Overall upload/download ratio
- **Average Speed**: Average speeds across all torrents
- **Peak Speed**: Maximum speeds achieved
- **Efficiency**: Download efficiency percentage
