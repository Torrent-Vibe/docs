# Speed Monitoring

Torrent Vibe provides comprehensive speed monitoring tools to track, analyze, and optimize your torrent download and upload speeds. This guide covers all speed monitoring features and optimization techniques.

## Speed Dashboard

### Current Speed Display

#### Main Speed Indicators
- **Download Speed**: Real-time download rate in MB/s or KB/s
- **Upload Speed**: Real-time upload rate in MB/s or KB/s
- **Combined Speed**: Total bandwidth usage
- **Speed Units**: Automatic unit conversion (B/s, KB/s, MB/s, GB/s)

#### Speed Context
- **Session Totals**: Total downloaded/uploaded this session
- **Daily Totals**: Today's download/upload totals
- **Monthly Usage**: Monthly bandwidth consumption
- **Speed Limits**: Current active speed limits

### Speed History Graphs

#### Real-time Graphs
- **Live Updates**: Graphs update every second
- **Time Scales**: 1 min, 5 min, 15 min, 1 hour, 24 hours
- **Interactive Zoom**: Click and drag to zoom into time periods
- **Multiple Metrics**: Display download, upload, and combined speeds

#### Historical Analysis
- **Speed Trends**: Speed patterns over time
- **Peak Analysis**: Identify peak speed periods
- **Average Calculations**: Calculate average speeds
- **Speed Distribution**: Speed frequency distribution
