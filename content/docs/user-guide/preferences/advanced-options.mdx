# Advanced Options

The Advanced Options section provides powerful configuration settings for experienced users who want fine-grained control over Torrent Vibe's behavior. This guide covers advanced configuration options and their implications.

## System Configuration

### Performance Tuning

#### CPU and Memory

##### Processing Threads
- **Thread Count**: Number of CPU threads to use
  - **Auto**: Automatically detect optimal thread count
  - **Manual**: Set specific thread count
  - **Per-Core**: One thread per CPU core
  - **Custom**: User-defined thread count

##### Memory Management
- **Cache Size**: Size of internal data cache
  - **Dynamic**: Adjust cache size automatically
  - **Fixed**: Set fixed cache size
  - **Percentage**: Cache as percentage of RAM
  - **Unlimited**: No cache size limit

##### Garbage Collection
- **GC Frequency**: How often to run garbage collection
- **GC Threshold**: Memory usage threshold for GC
- **GC Strategy**: Memory cleanup strategy
- **GC Logging**: Log garbage collection events

#### Disk I/O Optimization

##### Write Caching
- **Write Buffer Size**: Size of disk write buffer
- **Flush Interval**: How often to flush writes to disk
- **Sync Options**: Synchronous vs asynchronous writes
- **Cache Strategy**: Disk cache management strategy

##### Read Optimization
- **Read Ahead**: Pre-read data from disk
- **Buffer Pool**: Disk read buffer pool
- **Prefetching**: Intelligent data prefetching
- **Cache Warming**: Warm up disk cache on startup
