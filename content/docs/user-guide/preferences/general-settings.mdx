# General Settings

The General Settings section allows you to customize Torrent Vibe's overall behavior, appearance, and functionality. This guide covers all general configuration options.

## Interface Settings

### Appearance

#### Theme Selection
- **Light Theme**: Clean, bright interface for daytime use
- **Dark Theme**: Easy on the eyes, reduces eye strain
- **System Theme**: Automatically follows your OS theme
- **Custom Theme**: User-defined color schemes

#### Layout Options
- **Compact Mode**: Smaller interface elements, more content visible
- **Comfortable Mode**: Larger elements, more spacing
- **High Density**: Maximum information density
- **Touch Mode**: Optimized for touch devices

#### Language and Locale
- **Interface Language**: Choose display language
- **Date Format**: Select date display format
- **Time Format**: 12-hour or 24-hour time
- **Number Format**: Regional number formatting

### Display Preferences

#### Sidebar Configuration
- **Auto-hide Sidebar**: Hide sidebar when not in use
- **Sidebar Position**: Left or right side placement
- **Collapsed Groups**: Auto-collapse category/tag groups
- **Quick Access Items**: Customize sidebar quick access

#### Main View Options
- **Default View**: List, grid, or card view as default
- **Column Visibility**: Show/hide table columns
- **Sort Options**: Default sorting preferences
- **Group By**: Default grouping options

## Behavior Settings

### Startup Options

#### Application Startup
- **Launch on System Start**: Start with operating system
- **Minimize to Tray**: Start minimized to system tray
- **Check for Updates**: Automatically check for updates
- **Restore Previous Session**: Restore last session on startup

#### qBittorrent Connection
- **Auto-connect**: Automatically connect to qBittorrent
- **Connection Retry**: Retry failed connections
- **Reconnect Delay**: Delay between reconnection attempts
- **Connection Timeout**: Connection attempt timeout

### Notification Settings

#### Desktop Notifications
- **Enable Notifications**: Show desktop notifications
- **Notification Types**: Which events trigger notifications
- **Sound Alerts**: Play sounds with notifications
- **Notification Position**: Screen position for notifications

#### Notification Content
- **Completion Alerts**: Notify on torrent completion
- **Error Alerts**: Notify on torrent errors
- **Speed Alerts**: Notify on speed changes
- **System Alerts**: Notify on system events
