# Download Settings

The Download Settings section allows you to customize how Torrent Vibe handles torrent downloads, including paths, priorities, limits, and automation. This guide covers all download configuration options.

## Download Locations

### Default Paths

#### Primary Download Path
- **Default Directory**: Main download location for all torrents
- **Path Selection**: Browse and select download directory
- **Path Validation**: Verify path exists and is writable
- **Network Paths**: Support for network share paths

#### Path Templates
- **Variable Support**: Use variables in paths
  - `{category}`: Torrent category
  - `{date}`: Download date (YYYY-MM-DD)
  - `{year}`: Download year
  - `{month}`: Download month
- **Custom Variables**: Define your own variables
- **Path Preview**: Preview resolved paths

### Special Paths

#### Incomplete Downloads
- **Temporary Location**: Location for incomplete files
- **Auto-move**: Automatically move completed files
- **Cleanup Options**: Clean up temporary files
- **Space Management**: Manage temporary space usage

#### Watch Directory
- **Auto-add Directory**: Directory to monitor for new torrents
- **File Types**: Supported file types (.torrent, .magnet)
- **Scan Interval**: How often to scan the directory
- **Processing Rules**: Rules for processing found files
