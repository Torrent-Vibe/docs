### Priority Management

#### Torrent Priority
- **High Priority**: Download before others
- **Normal Priority**: Standard queue position
- **Low Priority**: Download after higher priority torrents
- **Skip**: Don't download this torrent

#### File Priority (Within Torrent)
- **High**: Download this file first
- **Normal**: Standard download order
- **Low**: Download after other files
- **Skip**: Don't download this file

## Monitoring and Statistics

### Torrent Progress

#### Progress Indicators
- **Progress Bar**: Visual download progress
- **Percentage Complete**: Numerical completion status
- **ETA Calculation**: Estimated time to completion
- **Pieces Information**: Downloaded pieces status

#### Detailed Statistics
- **Downloaded Amount**: Total bytes downloaded
- **Upload Amount**: Total bytes uploaded
- **Share Ratio**: Upload/download ratio
- **Time Active**: Total time torrent has been active

### Peer and Tracker Information

#### Peer Details
- **Connected Peers**: Number of active connections
- **Available Peers**: Total known peers in swarm
- **Seeders/Leechers**: Ratio of uploaders to downloaders
- **Peer List**: Detailed information about each peer

#### Tracker Status
- **Tracker List**: All trackers for this torrent
- **Working Trackers**: Successfully contacted trackers
- **Failed Trackers**: Trackers that are not responding
- **Update Intervals**: How often trackers are contacted

## Error Handling and Recovery

### Common Download Issues

#### Connection Problems
- **No Peers**: Torrent has no available peers
- **Slow Speeds**: Very low download speeds
- **Stalled Downloads**: Download stopped unexpectedly
- **Tracker Errors**: Trackers not responding

#### File Issues
- **Corrupted Files**: Downloaded files are damaged
- **Missing Files**: Some files not found
- **Permission Errors**: Cannot write to disk
- **Disk Full**: Insufficient storage space

### Recovery Options

#### Re-check Files
- **Full Re-check**: Verify all downloaded files
- **Partial Re-check**: Check specific files only
- **Resume from Point**: Continue from last good piece

#### Force Re-download
- **Re-download Corrupted**: Download damaged pieces again
- **Add More Trackers**: Improve peer discovery
- **Change Location**: Move to different storage
