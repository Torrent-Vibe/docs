# Torrent Details

Torrent Vibe provides comprehensive information about each torrent, allowing you to monitor progress, manage files, and control various aspects of the download process. This guide covers all the detailed information available for individual torrents.

## Accessing Torrent Details

### Methods to View Details

#### Click to Expand
- **List View**: Click the expand arrow (▶️) next to torrent name
- **Grid View**: Click the torrent card to expand details
- **Double-Click**: Double-click torrent name for full details

#### Context Menu
- **Right-Click**: Right-click torrent → "Properties"
- **Keyboard Shortcut**: Select torrent + Enter key
- **Menu Option**: Torrent menu → "Show Details"

#### Direct Navigation
- **URL Access**: Direct links to specific torrents
- **Bookmark**: Save links to important torrents
- **Share**: Share torrent detail links

## General Information Panel

### Basic Torrent Info

#### Identification
- **Name**: Full torrent name and title
- **Hash**: Unique SHA-1 hash identifier
- **Size**: Total size of all files
- **Pieces**: Number of pieces and piece size
- **Created**: Date and time torrent was created
- **Created By**: Software used to create torrent

#### Status Information
- **Status**: Current torrent state (downloading, seeding, etc.)
- **Progress**: Completion percentage and progress bar
- **ETA**: Estimated time to completion
- **Category**: Assigned category
- **Tags**: Applied tags
- **Priority**: Torrent priority level

### Transfer Statistics

#### Current Session
- **Downloaded**: Amount downloaded this session
- **Uploaded**: Amount uploaded this session
- **Download Speed**: Current download rate
- **Upload Speed**: Current upload rate
- **Share Ratio**: Upload/download ratio

#### All-Time Statistics
- **Total Downloaded**: Lifetime download amount
- **Total Uploaded**: Lifetime upload amount
- **Time Active**: Total time torrent has been active
- **Seeding Time**: Time spent seeding
- **Last Activity**: When torrent was last active

## File Information

### File List View

#### File Structure
- **Tree View**: Hierarchical display of files and folders
- **Flat View**: All files in a single list
- **Filtered View**: Show only specific file types

#### File Details
- **Name**: Full filename with path
- **Size**: Individual file size
- **Progress**: Download progress for each file
- **Priority**: Download priority (High/Normal/Low/Skip)
- **Availability**: Percentage of peers that have this file

### File Management

#### Priority Controls
- **Set High Priority**: Download this file first
- **Set Normal Priority**: Standard download priority
- **Set Low Priority**: Download after other files
- **Skip File**: Don't download this file
- **Auto-Priority**: Automatic priority based on file type

#### File Operations
- **Open File**: Open downloaded file with default application
- **Open Folder**: Open containing folder in file explorer
- **Copy Path**: Copy full file path to clipboard
- **Preview**: Preview file content (images, text, etc.)

## Peer Information

### Peer List

#### Peer Details
- **IP Address**: Peer's IP address and port
- **Client**: Torrent client software being used
- **Flags**: Connection status flags
- **Download Speed**: Speed at which this peer is downloading
- **Upload Speed**: Speed at which this peer is uploading
- **Downloaded**: Amount downloaded from this peer
- **Uploaded**: Amount uploaded to this peer

#### Peer Statistics
- **Connected Peers**: Number of active connections
- **Total Peers**: Total known peers in swarm
- **Seeds**: Number of peers with complete file
- **Leechers**: Number of peers still downloading
- **Peers with File**: Peers that have specific files

### Peer Management

#### Connection Controls
- **Ban Peer**: Block connection from specific peer
- **Kick Peer**: Disconnect from specific peer
- **Add Peer**: Manually add peer by IP:port
- **Copy Peer List**: Export peer information

#### Peer Filtering
- **Filter by Country**: Show peers from specific countries
- **Filter by Client**: Show specific torrent clients
- **Filter by Speed**: Show fast or slow peers
- **Encrypted Only**: Show only encrypted connections

## Tracker Information

### Tracker List

#### Tracker Details
- **URL**: Full tracker URL
- **Status**: Working, not working, or updating
- **Message**: Status message from tracker
- **Seeds**: Number of seeds reported by tracker
- **Leeches**: Number of leeches reported by tracker
- **Downloaded**: Number of completed downloads

#### Tracker Management
- **Add Tracker**: Add additional trackers
- **Remove Tracker**: Remove non-working trackers
- **Edit Tracker**: Modify tracker URL
- **Test Tracker**: Manually test tracker connectivity

### Tracker Statistics
- **Working Trackers**: Number of functional trackers
- **Total Trackers**: Total number of trackers
- **Last Update**: When trackers were last contacted
- **Next Update**: When next update is scheduled
- **Update Interval**: How often trackers are contacted

#### Advanced Controls

- **Timestamp**: When event occurred
