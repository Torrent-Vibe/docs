# Adding Torrents

Torrent Vibe provides multiple ways to add torrents to your download queue. This guide covers all available methods and their specific use cases.

## Methods to Add Torrents

### 1. Torrent Files (.torrent)

#### Drag and Drop
- **Simplest Method**: Drag `.torrent` files directly onto the interface
- **Batch Support**: Drop multiple files at once
- **Auto-Detection**: Files are automatically recognized and processed

#### File Browser
1. Click **‚ûï Add Torrent** button
2. Select **"From File"**
3. Browse and select `.torrent` files
4. Choose multiple files using Ctrl+Click (Windows/Linux) or Cmd+Click (macOS)

#### URL Upload
1. Click **‚ûï Add Torrent** button
2. Select **"From URL"**
3. Enter the direct URL to a `.torrent` file
4. Click **Add** to download and process

### 2. Magnet Links

#### Direct Paste
- **Copy magnet link** from any source
- **Paste directly** into Torrent Vibe's search bar
- **Auto-detection**: Magnet links are automatically recognized

#### Manual Entry
1. Click **‚ûï Add Torrent** button
2. Select **"Magnet Link"**
3. Paste the full magnet link
4. Click **Add**

#### Browser Integration
- **One-click adding** from supported browsers
- **Extension available** for Chrome, Firefox, and Edge
- **Context menu option** for links

## Add Torrent Dialog

When adding torrents, you'll see a comprehensive dialog with several sections:

### Basic Information
- **Torrent Name**: Auto-filled from torrent metadata
- **Size**: Total size of all files
- **Files**: Number of files in torrent
- **Comment**: Optional description from uploader

### Download Options

#### Save Path
- **Default Location**: Uses your configured default download folder
- **Custom Path**: Browse to select a different location
- **Subfolder**: Create torrent-specific subfolder
- **Path Templates**: Use variables like `{category}`, `{date}`

#### Category Assignment
- **Choose Category**: Select from existing categories
- **Create New**: Add a new category on the fly
- **Auto-Categorization**: Based on filename patterns

#### Tags
- **Apply Tags**: Select multiple tags
- **Create Tags**: Add new tags during addition
- **Tag Suggestions**: Based on torrent content

### File Selection

#### File List
- **Expand/Collapse**: View all files in the torrent
- **Checkboxes**: Select which files to download
- **File Sizes**: Individual file sizes
- **File Types**: Extensions and categories

#### Priority Settings
- **High Priority**: Download first
- **Normal Priority**: Standard download order
- **Low Priority**: Download after others
- **Skip**: Don't download this file

### Advanced Options

#### Torrent Options
- **Start Torrent**: Begin downloading immediately
- **Skip Hash Check**: Faster but less safe
- **Sequential Download**: Download files in order
- **First/Last Piece First**: Optimize for streaming

#### Bandwidth Limits
- **Download Limit**: Set per-torrent speed limit
- **Upload Limit**: Control upload speed
- **Alternative Limits**: Use when alternative speed limits are active

#### Connection Settings
- **Maximum Connections**: Per-torrent connection limit
- **Maximum Upload Slots**: Simultaneous upload connections
- **Share Ratio Limit**: Stop at specific ratio
- **Seed Time Limit**: Stop seeding after time/duration

## Batch Operations

### Adding Multiple Torrents

#### Bulk File Selection
1. Select multiple `.torrent` files
2. All files are processed together
3. Configure options for all torrents at once
4. Individual settings can be adjusted per torrent

#### Mass Magnet Links
- Paste multiple magnet links (one per line)
- Process all links simultaneously
- Apply same settings to all torrents

### Template Application
- **Save Settings**: Create templates for common scenarios
- **Apply Template**: Use saved settings for new torrents
- **Bulk Edit**: Modify settings for multiple selected torrents

## Error Handling

### Common Issues

#### "Torrent File Corrupted"
**Symptoms**: Error message about invalid torrent file
**Solutions**:
- Re-download the torrent file
- Check file integrity
- Try alternative sources

#### "Magnet Link Invalid"
**Symptoms**: Unable to parse magnet link
**Solutions**:
- Verify the magnet link format
- Check for truncated links
- Try copying the link again

#### "Insufficient Disk Space"
**Symptoms**: Warning about storage space
**Solutions**:
- Free up disk space
- Change download location
- Select fewer files to download

#### "Tracker Unavailable"
**Symptoms**: No peers found initially
**Solutions**:
- Wait for tracker update
- Add additional trackers
- Check torrent health on other sites

## Troubleshooting

### Adding Issues
- **Files not appearing**: Check file permissions
- **Slow processing**: Large torrents may take time
- **Network timeouts**: Check internet connection
- **Storage errors**: Verify write permissions

### Performance Issues
- **Slow downloads**: Check speed limits and connections
- **High CPU usage**: Reduce concurrent torrents
- **Memory issues**: Close other applications
- **Disk I/O**: Use faster storage if possible
