# Troubleshooting

This guide helps you resolve common issues with Torrent Vibe. Check here first when you encounter problems.

## Connection Issues

### Can't Connect to qBittorrent

**Symptoms**: "Connection failed" or "Unable to connect"

**Solutions**:
1. **Verify qBittorrent is running**
   - Check system tray or task manager
   - Start qBittorrent if not running

2. **Check Web UI settings**
   - Open qBittorrent → Tools → Preferences
   - Go to Web UI tab
   - Ensure "Enable Web User Interface" is checked

3. **Verify connection details**
   - Host: `localhost` or `127.0.0.1`
   - Port: `8080` (default)
   - Username/password match qBittorrent settings

4. **Test direct access**
   - Open browser to `http://localhost:8080`
   - Verify qBittorrent Web UI loads

### Authentication Failed

**Symptoms**: Login rejected despite correct credentials

**Solutions**:
1. **Check username/password**
   - Verify exact match with qBittorrent settings
   - Check for extra spaces or case sensitivity

2. **Reset qBittorrent credentials**
   - In qBittorrent: Tools → Preferences → Web UI
   - Change username/password
   - Restart qBittorrent

3. **Clear saved credentials**
   - In Torrent Vibe: Settings → Servers
   - Remove and re-add server connection

## Download Issues

### Torrents Not Downloading

**Symptoms**: Torrents stuck at 0% or very slow speeds

**Solutions**:
1. **Check peer connections**
   - View torrent details → Peers tab
   - Ensure peers are connected
   - Try forcing a recheck

2. **Verify tracker status**
   - Check tracker URLs are accessible
   - Add additional trackers if needed
   - Force tracker announce

3. **Check file permissions**
   - Ensure write access to download directory
   - Verify sufficient disk space
   - Check for file locks

### Slow Download Speeds

**Symptoms**: Downloads slower than expected

**Solutions**:
1. **Check internet connection**
   - Test speed with online speed test
   - Compare with expected speeds
   - Check for bandwidth limits

2. **Optimize settings**
   - Increase connection limits
   - Enable more peer discovery methods
   - Adjust speed limits if too restrictive

3. **Monitor peer quality**
   - Check peer upload speeds
   - Look for high-quality seeds
   - Consider torrent health

## Performance Issues

### High CPU/Memory Usage

**Symptoms**: Application uses excessive system resources

**Solutions**:
1. **Adjust settings**
   - Reduce maximum connections
   - Disable unnecessary features
   - Lower hash checking threads

2. **Monitor active torrents**
   - Limit concurrent downloads
   - Pause inactive torrents
   - Clean up completed torrents

3. **Check for conflicts**
   - Close other bandwidth-intensive applications
   - Update antivirus software
   - Check for malware

### Application Freezes or Crashes

**Symptoms**: Interface becomes unresponsive or crashes

**Solutions**:
1. **Restart application**
   - Close and reopen Torrent Vibe
   - Check for updates

2. **Clear cache and settings**
   - Reset to default settings
   - Clear application cache
   - Reconfigure from scratch

3. **Check system resources**
   - Monitor available RAM
   - Check disk space
   - Update system drivers

## Settings Issues

### Settings Not Saving

**Symptoms**: Configuration changes don't persist

**Solutions**:
1. **Check permissions**
   - Ensure write access to application directory
   - Run as administrator if needed

2. **Restart application**
   - Changes may require restart
   - Check after reopening

3. **Reset settings**
   - Use factory reset option
   - Reconfigure step by step

### Features Not Working

**Symptoms**: Specific features unavailable or malfunctioning

**Solutions**:
1. **Check prerequisites**
   - Verify required settings are enabled
   - Confirm API keys are valid
   - Test network connectivity

2. **Update application**
   - Check for newer versions
   - Install available updates
   - Review changelog for fixes

## File Management Issues

### File Association Problems

**Symptoms**: .torrent files don't open with Torrent Vibe

**Solutions**:
1. **Repair associations**
   - Settings → Preferences → Repair torrent file association
   - Restart computer
   - Test with a .torrent file

2. **Manual association**
   - Right-click .torrent file → Open with
   - Choose Torrent Vibe
   - Set as default

### Download Location Issues

**Symptoms**: Files downloading to wrong location

**Solutions**:
1. **Check save paths**
   - Verify default save path in settings
   - Check per-torrent save paths
   - Ensure paths exist and are writable

2. **Path mapping**
   - For multi-server setups
   - Configure correct path mappings
   - Test path accessibility

## Network Issues

### Firewall Blocking

**Symptoms**: Connection blocked by security software

**Solutions**:
1. **Add exceptions**
   - Allow Torrent Vibe through firewall
   - Add port 8080 to allowed ports
   - Configure antivirus exclusions

2. **Disable temporarily**
   - Temporarily disable firewall/antivirus
   - Test connection
   - Re-enable with proper exceptions

### Port Conflicts

**Symptoms**: Port already in use error

**Solutions**:
1. **Change ports**
   - Use different port in settings
   - Check for other applications using port
   - Use port scanner to find available ports

2. **Free up ports**
   - Close conflicting applications
   - Restart services
   - Check system processes
