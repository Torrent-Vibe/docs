# Monitoring Downloads

Torrent Vibe provides comprehensive monitoring tools to track your download progress, speeds, and overall torrent health.

## Real-time Statistics

### Speed Monitoring

**Current Speeds**
- **Download Speed**: Live download rate (KB/s, MB/s)
- **Upload Speed**: Live upload rate for seeding
- **Session Totals**: Total data transferred in current session

**Speed History**
- Graph of speed over time
- Peak speed tracking
- Average speed calculations

![Download Progress](/screenshots/TorrentVibeScreenshot_DownloadingProgress.jpeg)

### Progress Tracking

**Visual Indicators**
- Blue progress bars for each torrent
- Percentage completion
- Estimated time remaining (ETA)
- File size and downloaded amount

**Status Information**
- **Downloading**: Actively downloading
- **Seeding**: Sharing completed files
- **Paused**: Temporarily stopped
- **Completed**: Finished downloading
- **Error**: Download problems

## Detailed Torrent Information

### General Tab

**Basic Information**
- Torrent name and size

**Transfer Statistics**
- Total downloaded/uploaded

- Connected peers count

### Files Tab

**File List**
- Individual file names
- File sizes
- Download progress per file
- Priority settings
- Completion status

**File Management**
- Change file priorities
- Skip unwanted files
- View file locations
- Check file integrity

### Peers Tab

**Peer Information**
- Connected peer IP addresses
- Peer countries (if enabled)
- Download/upload speeds per peer
- Peer client software
- Connection status

**Peer Statistics**
- Total peer count
-Seeder/leecher ratio
- Peer connection quality
- Geographic distribution

### Trackers Tab

**Tracker Status**
- Tracker URLs
- Tracker status (working/error)
- Last announce time
- Next announce time
- Error messages

**Tracker Management**
- Add new trackers
- Remove broken trackers
- Edit tracker URLs
- Force announce
