title: Monitoring Downloads
description: Track and manage your torrent download progress
---

Track real-time download progress with detailed statistics and visual indicators.

## Progress Details

Overview of active downloads in the main interface.

![Download Progress](/screenshots/TorrentVibeScreenshot_DownloadingProgress.jpeg)

- **Status**: Current status, e.g., downloading, seeding, paused, error
- **Progress Bar**: Visual completion percentage with color-coded status
- **Size**: Total size and downloaded amount
- **ETA**: Estimated time remaining based on current speed

## Content Details

Monitor individual files within your torrents through the sidebar content tab.

![Sidebar Content Details](/screenshots/TorrentVibeScreenshot_Downloading_Sidebar_Contents.jpeg)

- **File List**: Complete list of all files in the torrent
- **File Sizes**: Individual file sizes and download progress
- **Priority Settings**: Set download priorities (High/Normal/Low)
- **Selective Download**: Choose which files to download

## Peer Connections

View and manage connected peers for optimal download performance.

![Sidebar Peers Information](/screenshots/TorrentVibeScreenshot_Downloading_Sidebar_Peers.jpeg)

- **Peer List**: All connected peers with IP addresses
- **Download/Upload Speeds**: Speed contribution from each peer
- **Geographic Location**: Peer locations (when available)
- **Client Information**: Torrent client software used by peers

## Tracker Status

Monitor tracker connectivity and performance metrics.

![Sidebar Trackers Information](/screenshots/TorrentVibeScreenshot_Downloading_Sidebar_Trackers.png)

- **Tracker URLs**: List of all configured tracker addresses
- **Connection Status**: Working/failed/error states for each Tracker
