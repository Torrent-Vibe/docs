## Basic Search

### Text Search

#### Simple Text Search
- **Single Words**: Search for torrents containing specific words
- **Multiple Words**: Search for torrents containing all words
- **Exact Phrases**: Use quotes for exact phrase matching
- **Case Insensitive**: Search is not case-sensitive

#### Search Examples
```
"movie"              # Torrents containing "movie"
"action movie"       # Torrents containing both "action" and "movie"
"Avengers Endgame"   # Torrents containing exact phrase
```

### Field-Specific Search

#### Torrent Name Search
- **Default Search**: Searches torrent names by default
- **Name Field**: `name:` prefix for explicit name search
- **Partial Matching**: Matches partial words and substrings

#### File Name Search
- **File Search**: `file:` prefix to search within files
- **Path Search**: `path:` prefix to search file paths
- **Content Search**: Search within file contents (limited)

## Advanced Search Syntax

### Boolean Operators

#### AND Operator
- **Default Behavior**: Multiple terms are AND-ed together
- **Explicit AND**: Use `AND` or `&&` between terms
- **Example**: `action AND movie` or `action movie`

#### OR Operator
- **Alternative Terms**: Use `OR` or `||` between terms
- **Example**: `action OR comedy` finds action OR comedy movies

#### NOT Operator
- **Exclusion**: Use `NOT` or `-` to exclude terms
- **Example**: `movie NOT comedy` finds movies that are not comedies

### Field Search

#### Supported Fields
```
name:         # Torrent name
file:         # File names within torrent
path:         # File paths
category:     # Category name
tag:          # Tag name
tracker:      # Tracker URL
hash:         # Torrent hash
size:         # Torrent size
status:       # Torrent status
ratio:        # Upload/download ratio
added:        # Date added
completed:    # Date completed
```

#### Field Examples
```
name:avengers           # Torrents with "avengers" in name
category:movies         # Torrents in "movies" category
tag:hd                  # Torrents with "hd" tag
size:>1GB              # Torrents larger than 1GB
added:>2024-01-01      # Torrents added after Jan 1, 2024
ratio:<1.0             # Torrents with ratio less than 1.0
```

### Wildcard Search

#### Wildcard Characters
- **Asterisk (*)**: Matches zero or more characters
- **Question Mark (?)**: Matches exactly one character
- **Examples**:
  ```
  movie*        # Matches "movie", "movies", "moviemaker"
  test?         # Matches "test1", "testA", but not "test12"
  *action*      # Matches any torrent containing "action"
  ```

### Range Search

#### Numeric Ranges
- **Greater Than**: `field:>value`
- **Less Than**: `field:<value`
- **Between**: `field:value1..value2`
- **Examples**:
  ```
  size:>500MB          # Larger than 500MB
  size:<10GB           # Smaller than 10GB
  size:1GB..5GB        # Between 1GB and 5GB
  ```

#### Date Ranges
- **Date Format**: `YYYY-MM-DD` or `YYYY-MM-DD HH:MM:SS`
- **Relative Dates**: `today`, `yesterday`, `this week`, etc.
- **Examples**:
  ```
  added:>2024-01-01    # Added after Jan 1, 2024
  completed:today      # Completed today
  added:this week      # Added this week
  ```
