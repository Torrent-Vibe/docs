Tags in Torrent Vibe provide a flexible, non-hierarchical way to organize and categorize your torrents. Unlike categories, tags allow multiple assignments and complex relationships, making them ideal for cross-cutting classifications.

## Understanding Tags

### What are Tags?

Tags are flexible labels that you can apply to torrents to:
- **Multiple Assignment**: One torrent can have many tags
- **Cross-Category**: Tags work across different categories
- **Dynamic Organization**: Easy to add, remove, and modify
- **Search and Filter**: Powerful filtering and search capabilities
- **Visual Identification**: Color-coded tag display

## Applying Tags to Torrents

### Manual Tagging

#### Single Torrent Tagging
1. **Select torrent** in main list
2. **Right-click** ‚Üí "Add Tags"
3. **Choose from list** or type to search
4. **Apply tags**

#### Batch Tagging
1. **Select multiple torrents** (Ctrl+Click or Shift+Click)
2. **Right-click** ‚Üí "Add Tags"
3. **Select tags** to apply to all selected torrents
4. **Choose mode**: Add to existing or replace all tags

### Automatic Tagging

#### Rule-Based Tagging
- **Filename Patterns**: Apply tags based on filename keywords
- **Size Ranges**: Tag based on torrent size
- **Tracker Sources**: Tag based on source tracker
- **File Types**: Tag based on content type

#### Smart Tagging
- **Content Analysis**: Analyze torrent content
- **Metadata Reading**: Use embedded metadata
- **User Learning**: Learn from manual tagging patterns
- **AI Suggestions**: Automatic tag suggestions

## Tag-Based Filtering and Search

### Tag Filtering

#### Basic Tag Filters
- **Single Tag**: Show torrents with specific tag
- **Multiple Tags (AND)**: Show torrents with all selected tags
- **Multiple Tags (OR)**: Show torrents with HD or Action tags
- **Exclude Tags**: Hide torrents with specific tags

#### Advanced Tag Filters
- **Tag Combinations**: Complex boolean logic
- **Tag Groups**: Saved filter combinations
- **Negative Filters**: Exclude certain tag combinations
- **Dynamic Filters**: Filters that update automatically

### Tag Search

#### Search Syntax
```
tag:HD                    # Torrents with HD tag
tag:HD+Action            # Torrents with both HD and Action tags
tag:HD|Action            # Torrents with HD or Action tags
-tag:Old                 # Exclude torrents with Old tag
tag:"High Quality"       # Tags with spaces (use quotes)
```

#### Search Integration
- **Global Search**: Tag search in main search bar
- **Quick Filters**: Tag-based quick filter buttons
- **Saved Searches**: Save complex tag searches
- **Search History**: Recently used tag searches
