Categories in Torrent Vibe help you organize and manage your torrents by grouping them based on content type, source, or any other criteria you choose. This guide covers creating, managing, and using categories effectively.

## Understanding Categories

### What are Categories?

Categories are organizational containers that help you:
- **Group Similar Content**: Movies, TV shows, music, software, etc.
- **Apply Consistent Settings**: Same download paths, speed limits, etc.
- **Quick Filtering**: Find torrents by category
- **Bulk Operations**: Apply actions to entire categories
- **Visual Organization**: Color-coded interface elements

## Creating Categories

### Basic Category Creation

#### Through Interface
1. **Open Category Manager**: Click the folder icon (üìÅ) in sidebar
2. **Click "Add Category"**: Green plus button
3. **Enter Details**:
   - **Name**: Category name (e.g., "Movies")
   - **Color**: Choose a color for visual identification
   - **Icon**: Select an icon (optional)

#### Quick Creation
- **Right-click torrent** ‚Üí "Create Category"
- **During torrent addition** ‚Üí "New Category" option
- **Import from file** ‚Üí Bulk category creation

### Advanced Category Setup

#### Category Properties
- **Display Name**: Full name shown in interface
- **Description**: Optional description
- **Color Scheme**: Primary and accent colors
- **Icon**: Custom icon or emoji
- **Parent Category**: Create hierarchical categories

#### Default Settings
- **Download Path**: Default save location for category
- **Speed Limits**: Category-specific speed limits
- **Auto-Management**: Automatic torrent assignment
- **Priority**: Category priority level

## Managing Categories

### Category List View

#### Organization Options
- **Sort by Name**: Alphabetical order
- **Sort by Count**: Number of torrents in category
- **Group by Type**: Group similar categories
- **Custom Order**: Drag and drop to reorder

#### Category Statistics
- **Torrent Count**: Number of torrents in category
- **Total Size**: Combined size of all torrents
- **Active Torrents**: Currently downloading/seeding
- **Completion Rate**: Average completion percentage

### Editing Categories

#### Basic Editing
1. **Right-click category** ‚Üí "Edit"
2. **Modify properties**:
   - Change name, color, icon
   - Update description
   - Adjust settings

#### Bulk Editing
- **Select multiple categories** using checkboxes
- **Apply changes** to all selected categories
- **Merge categories** to combine similar ones
- **Delete categories** and reassign torrents

## Assigning Torrents to Categories

### Manual Assignment

#### Single Torrent
1. **Select torrent** in main list
2. **Right-click** ‚Üí "Set Category"
3. **Choose from dropdown** or create new
4. **Apply changes**

#### Batch Assignment
1. **Select multiple torrents** (Ctrl+Click or Shift+Click)
2. **Right-click** ‚Üí "Set Category"
3. **Choose category** for all selected torrents
4. **Confirm assignment**

### Automatic Assignment

#### Rule-Based Assignment
- **Filename Patterns**: Assign based on filename keywords
- **Size Ranges**: Assign based on torrent size
- **Tracker URLs**: Assign based on source tracker
- **File Types**: Assign based on content type

#### Smart Assignment
- **Content Analysis**: Analyze torrent content
- **Metadata Reading**: Use torrent metadata
- **User History**: Learn from past assignments
- **AI Suggestions**: Automatic category suggestions

## Best Practices

### Organization Strategies

#### Content-Based Organization
- **Media Types**: Movies, TV, Music, Books
- **File Types**: Video, Audio, Software, Documents
- **Quality Levels**: HD, SD, 4K, etc.
- **Source Types**: Trusted sources, public trackers

#### Workflow-Based Organization
- **Priority Levels**: High, Normal, Low priority
- **Time Sensitivity**: Urgent, Normal, Archive
- **Project-Based**: Group by projects or clients
- **Seasonal**: Organize by time periods

### Naming Conventions

#### Consistent Naming
- **Standard Formats**: Use consistent category names
- **Abbreviations**: Use standard abbreviations
- **Language**: Choose primary language
- **Case Consistency**: Uppercase, lowercase, or title case

#### Hierarchical Naming
- **Parent/Child**: Use slashes for hierarchy (Movies/Action)
- **Numbering**: Use numbers for ordering (01-Movies, 02-TV)
- **Prefixes**: Use prefixes for grouping (A-Movies, B-TV)

### Maintenance

#### Regular Cleanup
- **Remove Empty**: Delete unused categories
- **Merge Similar**: Combine redundant categories
- **Update Rules**: Refresh assignment rules
- **Archive Old**: Move old categories to archive

#### Performance Optimization
- **Limit Categories**: Don't create too many categories
- **Balance Sizes**: Keep category sizes reasonable
- **Monitor Usage**: Track which categories are used most
- **Optimize Rules**: Keep assignment rules efficient

## Troubleshooting

### Common Issues

#### Assignment Problems
- **Auto-assignment not working**: Check rules and patterns
- **Wrong category assigned**: Review and correct rules
- **Categories not showing**: Check category visibility settings

#### Performance Issues
- **Slow category loading**: Too many torrents per category
- **Memory usage high**: Large number of categories
- **Interface lag**: Complex category hierarchies

#### Import/Export Issues
- **Import failures**: Check file format and permissions
- **Export incomplete**: Verify export settings
- **Template conflicts**: Resolve naming conflicts
