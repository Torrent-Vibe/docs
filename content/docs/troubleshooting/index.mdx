## Common Issues

### Connection Problems

If you're having trouble connecting Torrent Vibe to qBittorrent:

1. **Verify qBittorrent Web UI is enabled**
   - Open qBittorrent preferences
   - Go to "Web UI" section
   - Ensure "Web User Interface (Remote control)" is enabled
   - Set a username and password if required

2. **Check connection settings**
   - Verify the IP address and port are correct
   - Ensure qBittorrent is running
   - Check firewall settings

3. **Network connectivity**
   - Test if you can access qBittorrent Web UI directly in your browser
   - Verify your network allows the connection

### Performance Issues

For slow performance or high resource usage:

1. **Check system resources**
   - Monitor CPU and memory usage
   - Close unnecessary applications
   - Ensure adequate free disk space

2. **Optimize qBittorrent settings**
   - Adjust connection limits
   - Configure disk cache settings
   - Set appropriate speed limits

### Installation Issues

If you encounter problems during installation:

1. **Run as administrator (Windows)**
   - Right-click installer and select "Run as administrator"

2. **Check antivirus software**
   - Temporarily disable antivirus during installation
   - Add Torrent Vibe to antivirus exclusions

3. **Verify system requirements**
   - Ensure you meet minimum system requirements
   - Update your operating system

## Getting Help

If you can't find a solution here:

If you're still experiencing issues after trying these solutions, please:

1. Gather information about your setup:
   - Operating system and version
   - qBittorrent version
   - Torrent Vibe version
   - Error messages or screenshots

2. Check the [FAQ](./faq.mdx) for additional solutions
3. Contact support with the gathered information
